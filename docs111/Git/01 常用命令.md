#### 1. 获取所有远程分支
```bash
git fetch origin
```
运行结果：
```
* [new branch]      main       -> origin/main
```

这将从仓库中获取所有远程分支。
- `origin` 是要定位的远程名称。
>如果有一个上游远程名称 `upstream` ，可以直接调用 `git fetch upstream`


https://www.freecodecamp.org/chinese/news/git-checkout-remote-branch-tutorial/


### 更新远端分支
```bash
git remote update origin --prune
```
